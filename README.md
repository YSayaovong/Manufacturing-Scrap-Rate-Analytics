# Scrap Rate Analytics

This project provides a complete analytical view of manufacturing scrap behavior across multiple production lines, products, and defect categories. It models the flow of production and scrap data, computes core operational KPIs, and produces clear visual outputs for operational decision-making.

## What This Project Does
- Processes raw production and scrap records into structured KPI datasets.
- Calculates daily scrap performance, scrap rate by production line, and top defect drivers.
- Generates visual reports that highlight trends, outliers, and root‑cause patterns.
- Organizes data into a clean analytical model suitable for dashboards, operations meetings, and continuous improvement analysis.
- Supports both CSV-based and database-ready workflows for flexible deployment.

## Business Value
- **Reduces operational blind spots** by centralizing production and scrap metrics in a traceable analytical model.
- **Improves scrap visibility** by showing when, where, and why scrap increases across the factory.
- **Prioritizes quality issues** through defect Pareto analysis, helping leaders focus on the highest-impact problems.
- **Supports Lean/CI initiatives** by quantifying line performance and identifying process deviations.
- **Enhances decision-making** for supervisors, engineers, and production planners through clear trend and performance insights.
- **Strengthens reporting consistency** by automating KPI calculations and eliminating manual spreadsheet errors.
